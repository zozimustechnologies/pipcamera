# PiP Camera

A lightweight Chrome extension that opens your camera in picture-in-picture mode, allowing you to video chat or present while browsing.

## Features

- 📷 **Quick Camera Access** - Open your camera with a single click
- 🎥 **Picture-in-Picture Mode** - Float your camera feed on top of any window
- 🔒 **Privacy Focused** - Runs locally, no cloud uploads
- ⚡ **Lightweight** - Minimal resource usage
- 🎯 **Easy to Use** - Simple, intuitive interface

## Installation

1. Clone or download this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select this folder
5. The PiP Camera icon will appear in your Chrome toolbar

## Usage

1. Click the **PiP Camera** icon in your toolbar
2. Click **Start Camera** to begin capturing video
3. Click **Enter PiP** to open the camera in picture-in-picture mode
4. The floating window will appear and can be moved around
5. Click **Exit PiP** to return to normal view
6. Click **Stop Camera** when finished

## Requirements

- Google Chrome (or Chromium-based browser)
- Camera permission (you'll be prompted on first use)

## Permissions

This extension requests the following permissions:

- **camera** - To access your webcam
- **activeTab** - To work with the active tab
- **storage** - To remember settings
